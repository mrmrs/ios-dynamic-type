Dynamic Type in iOS

A css module for using dynamic type on iOS devices.

Install via NPM

npm install --save-dev ios-dynamic-type

Clone the repo

git clone git@github.com:mrmrs/ios-dynamic-type.git

Include via CDN

<link rel="stylesheet" href="https://npmcdn.com/ios-dynamic-type@1.0.0/css/ios-dynamic-type.min.css">
Download Source

"Going beyond the system font, iOS has dynamic type behavior, which can provide an additional level of fit and finish to your content. These text styles identify more than simply a particular font family; instead, they represent an entire style, including size and weight. These styles are therefore characterized by values given to the more-general font CSS property."

The ability to leverage user settings at the browser or system level within your application's design is pretty compelling. There is now a w3c discussion on standardizing a way to reference the system font.

Examples (Can only be seen on an iOS device)

Headers

font: -apple-system-headline;

This is a headline that is quite long with a lot of letters and words

font: -apple-system-subheadline;

This is a subheadline that is also quite long with many characters

font: -apple-system-short-headline;

Short headline

font: -apple-system-short-subheadline;

Short subheadline

Body Copy

font: -apple-system-body;

You can say, "I love you," in Helvetica. And you can say it with Helvetica Extra Light if you want to be really fancy. Or you can say it with the Extra Bold if it's really intensive and passionate, you know, and it might work.

- Robert Bringhurst

font: -apple-system-tall-body;

Space in typography is like time in music. It is infinitely divisible, but a few proportional intervals can be much more useful than a limitless choice of arbitrary quantities.

- Robert Bringhurst

font: -apple-system-short-body;

Typographic style is founded not on any one technology of typesetting or printing, but on the primitive yet subtle craft of writing.

- Robert Bringhurst

Captions & Footnotes

font: -apple-system-footnote;

This is a footnote about how excited I am for more platform specific system fonts.

font: -apple-system-caption;

This is first caption style for something about how cute cats are

font: -apple-system-caption2;

This is the second caption style for something about how cute cats are
